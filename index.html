🔄 債務週期的迴圈與先後順序

每一次的經濟衰退到繁榮，皆是由債務（信貸）的擴張與收縮所驅動。儘管中央銀行的極端政策可能縮短或拉長某階段，但整體演進依然有跡可循：

早期復甦 ➔ 泡沫期 ➔ 頂部 ➔ 蕭條期 (可分歧為: 和諧去槓桿 或 通膨型蕭條) ➔ 正常化
在這個循環中，信貸是經濟的引擎。當信貸擴張快於產出時即形成「泡沫」，當信貸無以為繼時即觸發「頂部」與「蕭條」。

階段判定邏輯與投資策略

  • 1. 早期復甦 (Early Recovery): 實質 GDP 強勁反彈 (> 2%)。長端殖利率小於名目 GDP 增速，且殖利率曲線呈正斜率。(加碼股票、減碼現金)
  • 2. 泡沫期 (Bubble): M2 貨幣成長率顯著高出實質經濟增長。債務比急速攀升。(股票微調降、加碼黃金)
  • 3. 頂部 (Top): 殖利率曲線倒掛 (10Y-2Y < 0)。利率高於經濟增長。(加碼現金與公債避險、大砍股票)
  • 4. 蕭條期 (Depression): 實質 GDP 急墜。信用利差劇烈擴張。(加碼長天期公債 TLT 賺取資本利得、減碼股票)

    🚨 通膨型蕭條分支: 若蕭條期通膨過熱，須暴力加碼黃金與抗通膨債，防禦法幣崩潰。
  • 5. 和諧去槓桿 (Beautiful Deleveraging): M2 供給大於 GDP 增長，通膨維持溫和 2~4%，重拾正增長。(加碼黃金與股票)
  • 6. 正常化 (Normalization): 無異象，遵循 All Weather 黃金比例。

🗂️ 經濟四象限與判斷條件

市場無時無刻不處於四個象限之一。系統藉由 FRED API 實時抓取的 CPI 與 GDP 進行精確打點。

  • 1. 高增長 / 低通脹 (Goldilocks 金髮女孩經濟)
    條件：GDP YoY > 2.5%, CPI < 3.0%
    配置：提升(↑)股票與公債、降低(↓)黃金與商品。
  • 2. 高增長 / 高通脹 (Overheating 經濟過熱)
    條件：GDP YoY > 2.5%, CPI > 4.5%
    配置：提升(↑)大宗商品與黃金、降低(↓)長天期公債。
  • 3. 低增長 / 低通脹 (Deflationary Bust 通縮性衰退)
    條件：GDP YoY < 1.5%, CPI < 2.5%
    配置：提升(↑)長中天期公債與現金、降低(↓)股票。
  • 4. 低增長 / 高通脹 (Stagflation 停滯性通膨)
    條件：GDP YoY < 1.5%, CPI > 3.5%
    配置：提升(↑)黃金與短期現金(避開存續期風險)、降低(↓)股票與長天期公債。
⚠️ 數據滯後防禦: 若總體經濟數據延遲發布超過 30 天，系統底層會自動改採「移動平均」進行估值，並給出 ⚠️ 標籤確保不失真。
